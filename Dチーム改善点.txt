index.htmlの文字

画像たちのフォルダ名、ファイル名をすべてアルファベットと記号だけ（日本語を使わない名前）に変更
↓
それにともなってデータベースの画像パスの部分もすべてそろえる

DBのproductテーブルimage_path列には画像のファイル名のみに（例：Cookie.jpg）

FIC_D_CAREERDESIGN
	|

		|

		|	|
		|	|----Cookie.jpg
		|
		|

		|	|

		|
		|
		|----hot_drink
		|	|
		|	|----LemonTea.jpg
